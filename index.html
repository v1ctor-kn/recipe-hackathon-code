How to use

  1. Type ingredients you have (commas allowed).
  2. Optionally select dietary filters.
  3. Click "Find Recipes". Results show a short summary, ingredients used and a rough cook time.

What this app does

This frontend collects your input and delegates searching to the connected recipe engine (local script or API). Make sure script.js implements the getRecipes() function and reads filters by element IDs.
